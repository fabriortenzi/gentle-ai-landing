# gentle-ai Landing Page

Landing page for [gentle-ai](https://github.com/Gentleman-Programming/gentle-ai) — the Gentleman AI ecosystem configurator.

## Stack

- **Astro 6** — Static site generation
- **Tailwind CSS 4** — Styling
- **Vercel** — Hosting (zero-config static deploy)

## Development

```sh
npm install
npm run dev       # localhost:4321
npm run build     # production build → ./dist/
```
